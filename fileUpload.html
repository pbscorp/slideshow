Select media files from your local storage 📁 to be staged for uploading,

then select Upload Files  to load them into your workspace.
Once loaded, the media list will be refreshed with the added files.

 

An image file named ‘default’ will be used as the background for the slideshow.

 

When the slideshow is presented, the files with base names that match the names on the slide text will be shown and/or heard when the matching slide name is processed in the slideshow. The files extension will determine the media type.

Filenames that do not match slide text names will be ignored are highlighted in red text color and can be renamed.
